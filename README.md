Lightweight, client-side Single Page Application designed to solve the "availability tetris" problem. By ingesting standard `.ics` calendar files (common in Outlook, Google, Apple), or web links. It algorithmically determines optimal meeting slots. Unlike binary "free/busy" tools, it provides a fuzzy availability score, ranking time slots based on how many participants can attend.

Live : https://b2renger.github.io/CalSnap/

Local :

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
